I haven't written much in this file because tags are exceedingly simple, so this is a good time to talk about backlinks. Oftentimes, the links pointing to a note will say more about that note than the note itself. This is definitely true in this case; if you want to know where tags are being used and how they are being used, simply look at the backlinks and you will discover many other notes that link to and use tags. 


## Using tags to pinpoint specific parts of our notes 

I have described how tags can be used to describe classes and objects, but this is only one use case for tags, and it's not even something that tags are strictly needed for (as you can achieve the same thing with text [[metadata]]). The thing that tags show their real value for is pinpointing a specific place in a note. 

This exact concept is explained once again by this fantastic video: [[Obsidian Bases--Obsidian's Biggest Upgrade (Complete Guide)]] (watch at `48:42`).
