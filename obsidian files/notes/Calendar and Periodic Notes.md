


The **Calendar** and **Periodic Notes** community plugins go perfectly together. 


## Calendar
N.B.As I stated in [[Navigation and workspaces]], I put the Calendar view in the bottom right hand of my screen with a few other views to toggle between. 

---

The calendar plugin is a simple way of seeing the days which have notes, and how many words are written for these days. This same functionality also extends to weekly notes (and maybe monthly and yearly notes as well, but I don't personally use these). 

The plugin also creates a simple functionality for creating daily notes for days that are note today, which can come in extremely handy. 


## Periodic Notes

N.B. Thanks to Periodic Notes, you can turn off the [[core plugins|core plugin]] **Daily notes** (as it is redundant and will just clutter things).

---


The Periodic Notes plugin is an expansion on the Daily notes core plugin. Periodic notes allows for the creation of weekly, monthly, and even yearly notes. 


## How to use


In order to setup Periodic Notes, you may, if you desire, create a template for daily notes and select this template in the Periodic Notes settings. Also, you must create a folder for your daily notes. You may, then, repeat this process for weeks, months, and years if you so desire. 

![[Calendar and Periodic Notes.png]]

In my personal base, I have a "journal" folder, which contains a "daily" and a "weekly" folder (although I have admittedly never created a weekly note).
