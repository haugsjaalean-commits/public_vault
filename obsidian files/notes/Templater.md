

Templater is simply a syntax for creating different outcomes in templates based on things like the date, etc. This means that you do not have to know Templater to get a lot of value out of templates, but the chances are that, if you spend enough time in Obsidian, you will eventually find yourself wanting to use this plugin, in which case you can come back here to discover what it's capable of. 

Templater is a way to make [[templates]] more powerful. I suggest watching [[Automate Your Vault With Templater - How to Use Templater in Obsidian]] to get started. If I'm being honest, whenever I need to use templater, I simply ask AI to do it for me, because I don't actually use it enough for learning it to be worth it. I really don't use templater that often, but I can definitely imagine workflows where it is *extremely* useful.
