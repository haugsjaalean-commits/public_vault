# Loose rules to explain the functioning of my vault


This vault functions with the idea that every note is a discrete object. Each object has attributes. These attributes fall into two categories:
1. **Special attributes** are attributes whose values give important information which informs the nature of the individual object. 
2. **Benign attributes** are attributes that might enrich the use of the object, but do not define the individual object compared to its brethren. 

All objects inherit from a class, and all classes are defined by their unique tag. All class tags start with `#object`. 

In order to keep track of my different classes, I have created a *classes* folder with this path `obsidian files/classes`. Inside of this folder, I put all of my classes, which all inherit from the base class [[object]].
