In [[Modeling classes]], I talked about how we can use [[YAML|frontmatter]] to model our own objects and classes, but, once we've decided on how we want our classes to look, we will need a way to automate their creation. This is one of the main purposes of templates. I create templates for each different type of class I have.


## How to use


In order to set up templates, you must create a templates folder in your vault. In the settings, under templates (in core plugins), you must select that folder as the template folder:
