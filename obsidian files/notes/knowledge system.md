A **knowledge system** (KS) needs to be able to do many things at once. This makes the creation of one very difficult.

The first function (and challenge) of a KS is that it needs to be able to create the units of knowledge. In order to create a KS, one needs to think of a method for describing units in a way that is efficient, and makes sense for the human mind.

The second function of a KS is that it needs to be able to restructure the KG (Knowledge Graph) at any moment and any level, and allow the user to move freely through the KG.

These two things alone pose many problems, and I think that one could devote their life to creating the perfect system. As of right now, we have nothing that comes even a little bit close to the thing that I'm describing.
