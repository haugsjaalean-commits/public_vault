# How it works

> [!info]+ Definition of **operations of organization**
> An **operation of organization** is when we take a set $U$ of KIs and organize them.

Multi-level organization is when we create $n$ duplicates of $U$ ($U_1$) and execute an OO on all of them. These duplicates are numbered in the following manner: $U_{1.2}$, $U_{1.3}$, $U_{1.4}$, ... , $U_{1.n}$.

We then create $U_2$, where its set of KIs are all of the GPs of the duplicates of $U_1$.

We repeat this process $m$ times.

# Theories

I believe that, by using consecutive OOs we can achieve any and all desired structures of information.
