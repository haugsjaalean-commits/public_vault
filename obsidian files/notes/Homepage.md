

The home page is crucial for making the structure of the vault work. This addon allows us to come back to our *root note* automatically or with the click of a button. 

## How to setup
![[Homepage.png]]



In order to setup Homepage, all you need to do is select the file you wish to be the root file. In my case, I have called it "MOC" (for "Map Of Content"). 


## How to use

The homepage can be opened using multiple different methods. For starters, there is a button in the toolbar which will open it, but I prefer the hotkey I made for it (`Alt + H`). I also have the Homepage open when there are no other files open. 

There are many different functionalities of this addon, so I suggest you check them out if you're interested.
